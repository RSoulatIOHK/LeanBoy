A Game Boy / Game Boy Color emulator written in [Lean 4](https://lean-lang.org/).

## Quick Start

```sh
cd LeanBoy
lake build
.lake/build/bin/leanboy <rom.gb|rom.gbc>
```

See [LeanBoy/README.md](LeanBoy/README.md) for full build instructions, controls, and project structure.
